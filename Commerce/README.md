## 第六章·商业之美

《激荡三十年》是我读的关于商业，关于企业发展与经济规律，并让我喜欢商业之美的第一本书。因此，本章主要记录关于商业的各种读书笔记，亦或关于吴晓波栏目的学习所得。

## 阅读书单：

  * 《激荡三十年》
  * 《跌荡一百年》
  * 《浩荡两千年》
  * 《历代经历变革得失》
  * 《大败局》
  * 《把生命浪费在美好的事物上》
  * 《腾讯传》
