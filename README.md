# 一个人的活法


一点生活的感想，一点读书的乐趣。**知识改变命运，思维改变生活**。

这本书是自我成长的个人记录。书中的内容并不固定，但大都是关于读书，关于成长，关于思维，关于学习的方方面面。内容的素材也相对广泛，或是网上的付费阅读产品，或是自选书籍，亦或网上的博文或者书籍。
